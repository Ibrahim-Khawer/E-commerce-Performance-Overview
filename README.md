# E-commerce Performance Overview

## Business Problem
E-commerce businesses need to understand customer shopping patterns to optimize marketing strategies and maximize revenue. This project analyzes transactional data to uncover insights about customer behavior and product performance.

## Project Details
- **Dataset**: [Customer Shopping Behavior](https://github.com/Ibrahim-Khawer/E-commerce-Performance-Overview/blob/main/customer_shopping_behavior.csv) - 3,900 transactions, 18 features
- **Tools**: Python (Pandas, Matplotlib), Power BI
- **Dashboard**: [View Interactive Dashboard](https://github.com/Ibrahim-Khawer/E-commerce-Performance-Overview/blob/main/E-commerce%20Performance%20Overview.pdf)

## Data Preparation
- **Data Cleaning**: Handled missing review ratings using median imputation
- **Feature Engineering**: Created age groups and standardized purchase frequencies
- **Data Validation**: Ensured data consistency and prepared for analysis

## Analysis Highlights
- **Customer Segmentation**: New, Returning, and Loyal customers based on purchase history
- **Revenue Analysis**: Total revenue by gender, age groups, and subscription status
- **Product Performance**: Top-rated products and category-wise bestsellers
- **Discount Strategy**: Identified discount-dependent products and high-spending discount users

## Key Insights
- Subscribers spend significantly more than non-subscribers
- Clear customer segments with distinct spending behaviors
- Certain products are highly discount-dependent
- Express shipping users have higher average order values

## Business Recommendation
1. Enhance subscriber benefits to increase conversion
2. Develop targeted marketing for high-value customer segments
3. Re-evaluate discount strategy for profit protection
4. Implement loyalty programs to retain valuable customers
